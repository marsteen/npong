The game:


  Protect the triangles against the thiefs. When the last triangle 
  is gone, the game is over.
  
  You can play alone (singleplayer) or together with a friend (co-op).

  Press 1 to start singlerplayer-game.
  Press 2 to start co-op game.


**********************************************************************

Player 1: 


  Move forward with right mousebutton
  Fire with left mousebutton
  Move backward with middle mousebutton
  
  OR use the keyboard:
  
  Arrowkeys  = Move
  Delete-key = Fire
   

**********************************************************************
  
Player 2: 

  W A S D    = Move
  Tab-key    = Fire

**********************************************************************

Commandline-options:

 -w800x600       = window-mode 800x600 pixel
 -w1024x768      = window-mode 1024x768 pixel
 -w1280x1024     = window-mode 1280x1024 pixel
 -w1440x900      = window-mode 1440x900 pixel
 -w1920x1080     = window-mode 1920x1080 pixel
 -w3840x2160     = window-mode 3840x2160 pixel
 -linewith <v>   = set linewidth to v (default=1.0)
 -scale <v>      = set scalefactor to v (default=1.0)
 -mousespeed <v> = set mousespeed to v (default=10.0)
 
By default, the game starts in fullscreen-mode (without changing 
the resolution).
